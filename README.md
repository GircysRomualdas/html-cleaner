# HTML Cleaner

A web-based tool to clean, format, compress, preview HTML, and convert it to markdown.

---

## Live Demo

Try it here:

https://gircysromualdas.github.io/html-cleaner/

---

## Installation 

1. Clone the repository.

2. Install dependencies:
```bash
bun install
```

---

## Build

```bash
bun run build
```

---

## Run Locally

```bash
open index.html
```
